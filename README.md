## 🌊 Overview

A premium, cinematic portfolio experience built for modern digital architects. This project features a deep-sea atmosphere, fluid animations, and a high-performance interactive UI designed to showcase technical mastery and creative vision.

## ✨ Key Features

- **Cinematic Hero Experience**: High-impact visual landing with dynamic atmosphere and animated typography.
- **Projects Manifesto**: A curated showcase of featured work with high-fidelity interactions.
- **Mastery & Tech Stack**: Smooth, infinite marquee displaying core competencies and technologies.
- **Certificates Showcase**: Organized presentation of professional certifications and achievements.
- **Real-Time Data**: Seamless synchronization between the frontend and a persistent storage layer.
- **Responsive Architecture**: Fully optimized for a premium experience across all devices.

## 🛠️ Tech Stack

- **Frontend**: [React.js](https://reactjs.org/), [Framer Motion](https://www.framer.com/motion/), [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Backend**: [Node.js](https://nodejs.org/), [Express](https://expressjs.com/)
- **Database**: [SQLite](https://www.sqlite.org/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Recommended: Latest LTS)

### Installation

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   cd yasvanth-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment**
   Set up your environment variables in a `.env` file based on the provided template.

4. **Launch Development Server**
   ```bash
   npm run dev
   ```
